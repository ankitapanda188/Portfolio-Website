# 🌐 Personal Portfolio Website
A modern and responsive personal portfolio website developed using **HTML5** and **CSS3**. This portfolio showcases my education, technical skills, projects, certifications, internship experience, and contact information in a clean and professional layout.

## ✨ Features
- Modern and responsive design
- Fixed navigation bar
- Hero section with introduction
- About Me section
- Education timeline
- Skills and Interests cards
- Featured project showcase
- Internship and Certifications
- Contact section with social links
- Smooth scrolling and hover animations
- Mobile-friendly layout

## 🛠️ Technologies Used
- HTML5
- CSS3
- Font Awesome
- Google Fonts (Poppins)

## 📂 Project Structure
```
Portfolio/
```

## 🚀 Getting Started
1. Clone the repository
```bash
git clone https://github.com/yourusername/portfolio.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional installation or dependencies are required.
